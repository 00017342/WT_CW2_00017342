# 📚iReview

## 🧐 Project philosophy
    iReview is a book shelf right inside your web. Create, read, update, delete, rate your books inside iReview. If you were interested in books, It is definitely your best choice!

## 👨‍💻 Tech stack
### Languages:

    Html
    CSS
    Javascript
    JSON

### Technologies:
    
    Node.js
    Express
    Express-validator
    Express-ejs-layouts
    EJS
    Bootstrap

## 🛠️Install packages
```bash
npm install
```

## 📍Run locally
```bash
npm run dev
```
